## First Approach : Using 3 nested loops and check each subarray sum

```java
class Solution {
    public int maxSubArray(int[] nums) {
        if(nums.length==1){
            return nums[0];
        }
        int n = nums.length, maxsum=Integer.MIN_VALUE;
        for(int i =0; i<n; i++){
            for(int j =i; j<n; j++){
                int current_sum =0;
                for(int k=i; k<=j; k++){
                    current_sum += nums[k];
                }
                if(current_sum>maxsum){
                    maxsum = current_sum;
                }
            }
        }
        return maxsum;
    }
}
```
Time Complexity : O(n<sup>3</sup>)<br>
Space Complexity : O(1)

## Second Approach : Using 2 nested loops and keeping track of each element sum

```java
class Solution {
    public int maxSubArray(int[] nums) {
        if (nums.length == 1) {
            return nums[0];
        }
        int n = nums.length, maxsum = Integer.MIN_VALUE;
        for (int i = 0; i < n; i++) {
            int current_sum = 0;
            for (int j = i; j < n; j++) {
                current_sum += nums[j];
                if (current_sum > maxsum) {
                    maxsum = current_sum;
                }
            }
        }
        return maxsum;
    }
}
```
Time Complexity : O(n<sup>2</sup>)<br>
Space Complexity : O(1)
